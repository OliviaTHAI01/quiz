# 💖 แบบทดสอบบุคลิกภาพที่ซ่อนอยู่ (Psychology Quiz)

แบบทดสอบจิตวิทยาฉบับน่ารักที่ออกแบบมาเพื่อวิเคราะห์บุคลิกภาพในด้านการเป็นผู้นำและการเป็นผู้ตาม (Seme/Uke) ผ่านสถานการณ์จำลองในชีวิตประจำวัน โดยใช้ดีไซน์ที่เน้นความนุ่มนวลและเป็นมิตร

## ✨ ฟีเจอร์เด่น
- **Cute UI/UX**: ดีไซน์สไตล์พาสเทล พร้อมฟอนต์ 'Mali' ที่ดูเป็นกันเอง
- **Smart Randomization**: ระบบสลับข้อคำถามและสลับตัวเลือกทุกครั้งที่เล่นใหม่
- **Custom Cursor**: เมาส์รูปนิ้วมือสุดน่ารัก (SVG Custom Cursor)
- **Smooth Animations**: หัวใจเต้น (Heartbeat) และการเปลี่ยนหน้าแบบนุ่มนวล
- **Social Sharing**: ระบบแชร์ผลลัพธ์ที่จัดรูปแบบข้อความมาให้สวยงาม พร้อมส่งต่อในโซเชียลมีเดีย
- **Responsive Design**: ใช้งานได้ดีทั้งบนคอมพิวเตอร์และมือถือ

## 🛠️ เทคโนโลยีที่ใช้
- **HTML5 & CSS3** (Tailwind CSS via CDN)
- **JavaScript (Vanilla)**
- **Google Fonts** (Mali)
- **Lucide/Heroicons** (SVG Icons)

## 🚀 วิธีการใช้งาน
1. Download หรือ Clone โปรเจกต์นี้ไปที่เครื่องของคุณ
2. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
3. เริ่มทำแบบทดสอบให้ครบ 10 ข้อเพื่อดูผลลัพธ์!
